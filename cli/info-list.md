### inpack info-list

``` shell
inpack info-list --repo test
Found 19

| Name            | Last Version | Num | Updated    |

| nginx           | 1.18.0       | 2   | 2020-10-01 |
| postgresql      | 9.6.9        | 1   | 2020-10-01 |
| mysql57         | 5.7.28       | 1   | 2020-10-01 |
+-----------------+--------------+-----+------------+
```
